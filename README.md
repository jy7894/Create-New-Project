# Create-New-Project

A simple script to quickly set up a new project with a main file, a run script, and (for Python) a virtual environment.

---

## Supported Languages
- Python
- Java
- C++

---

## Usage

### 1. run the main script using:

#### Linux/MacOs
```bash
Create-New-Project {ProjectName} {LanguagePrefix} {PathToProject<Optional>}
```
#### Windows
```cmd
run.bat {ProjectName} {LanguagePrefix} {PathToProject<Optional>}
```

### 2. then to run the project use (inside the project folder)

#### Linux / MacOS
```bash
./run
```
#### Windows
```cmd
run.bat
```

# Create New Project

## Requirements
- **Python 3** (version 3.8 or higher recommended)
- **Git** (for cloning the repository)

---

## Download & Installation

### Linux/MacOS
```bash
git clone https://github.com/jy7894/Create-New-Project
cd Create-New-Project
./install.sh
```

### Windows
```cmd
git clone https://github.com/jy7894/Create-New-Project
cd Create-New-Project
```
