# Create-New-Project

A simple script to quickly set up a new project with a main file, a run script, and (for Python) a virtual environment.

---

## Supported Languages
- Python
- Java
- C++

---

## Usage

### 1. run the main script using your Python interpreter:

```cmd
(py/python/python3) Create-New-Project/src/main.py {ProjectName} {LanguagePrefix} {PathToProject}
```

### 2. run with the run script
#### Linux
```bash
sh run.sh
```
#### Windows
```cmd
run.bat
```
